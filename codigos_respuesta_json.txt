------------- Estos son los códigos de respuesta de los JSON y su significado -------------------

0 -> Password incorrecto
1 -> Email no exitente en base de datos
2 -> El correo ya existe en base de datos
3 -> La petición ha sido realizada satisfactoriamente
4 -> ID no existente en base de datos
5 -> Ingrese una petición en la URL de forma domicio.com/peticion
6 -> Ingrese una peticion valida
7 -> Uno o varios de los nombres de las columnas no coinciden con la base de datos
8 -> El usuario no esta activo, valide el codigo de verificacion.
9 -> El código de verificación no coincide.
10-> El teléfono ya existe en uno de los contactos de emergencia.
11-> Petición POST fallida.
12-> Petición DELETE fallida, no se puede eliminar por integridad de la base de datos 
13-> Hay un documento con el mismo nombre en la carpeta que intentas cargar
14-> Código de activacion no esta activo o no existe en base de datos
15-> El usuario ya tiene un plan activo, no se pueden tener dos planes al tiempo
16-> El usuario no tiene un plan activo
17-> El usuario no tiene plan activado
18-> La zona ya tiene este Servicio
19-> Codigo de zona invalido o vencido
20-> La zona no tiene establecimientos registrados
22-> El token a expirado
23-> El usuario no esta autorizado
24-> Autorización Requerida
25-> Credeciales Incorrectas
26-> No hay Alertas en la zona
27-> El cliente esta tratando de reaccionar a su propia alerta
28-> la alerta ya fue atendida por otro cliente
